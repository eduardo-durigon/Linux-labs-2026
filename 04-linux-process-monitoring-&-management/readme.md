🧪 Lab 04 – Linux Process Monitoring & Management

🎯 Objective
Learn how to monitor, analyze, and manage processes in a Linux environment, including understanding user permission boundaries.

🛠️ What I Did
Viewed running processes using ps aux and top
Filtered processes with grep
Simulated a background process using sleep
Investigated process details using ps -p
Terminated processes using kill
Tested permission limits by attempting to kill another user’s process and using sudo

🧠 Key Takeaways
Every process has a unique PID used for monitoring and control
ps provides a snapshot, while top shows real-time activity
Not all processes can be terminated without proper permissions
Process management is a fundamental skill for IT and cybersecurity

📌 Reflection
This lab helped me understand how to identify and manage processes in Linux, as well as the importance of permissions when performing system-level actions.
